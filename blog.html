نصب rubjs در گوشی موبایل

۴ خرداد ۱۴۰۴ · خواندن 3 دقیقه
Hadi Rostami
Front End Engineer

سلام! در این مقاله قصد داریم به شما آموزش دهیم چگونه در گوشی موبایل خود محیط برنامه‌نویسی راه‌اندازی کرده و با rubjs شروع به کدنویسی کنید. بهترین روش برای این کار استفاده از Termux است که یک شبیه‌ساز ترمینال لینوکس برای اندروید می‌باشد.

مرحله ۱: نصب Termux​

Termux یک اپلیکیشن رایگان و سبک است که به شما امکان اجرای دستورات لینوکس را در اندروید می‌دهد. برای نصب:

  • به Google Play Store یا F-Droid مراجعه کنید و Termux را دانلود و نصب کنید.

  • بعد از نصب، برنامه را باز کنید تا آماده استفاده شود.

مرحله ۲: آپدیت پکیج‌ها و نصب Node.js​

قبل از نصب هر چیزی بهتر است مخازن و پکیج‌های Termux را آپدیت کنید تا آخرین نسخه‌ها را داشته باشید:

pkg update && pkg upgrade -y

این دستور باعث می‌شود که فهرست بسته‌ها و نرم‌افزارهای نصب شده به‌روز شود.

حالا Node.js را نصب کنید که محیط اجرای JavaScript و npm را فراهم می‌کند:

pkg install nodejs -y

برای اطمینان از نصب درست Node.js، نسخه آن را بررسی کنید:

node -v
npm -v

مرحله ۳: نصب rubjs​

با داشتن Node.js، حالا می‌توانید rubjs را نصب کنید. این ابزار به شما امکان می‌دهد که پروژه‌های خود را با آن بسازید و اجرا کنید.

برای نصب rubjs به صورت سراسری (global) از دستور زیر استفاده کنید:

npm install -g rubjs

مرحله ۴: شروع به کدنویسی​

  • بعد از نصب موفقیت‌آمیز rubjs، می‌توانید کدنویسی را شروع کنید.
  • کدهای جاوااسکریپت خود را بنویسید و اجرا کنید.
  • تمام کارهایتان را در Termux و موبایلتان مدیریت کنید.

مرحله ۵: اجرای ربات​

برای اجرای ربات از دستور node filename.js استفاده کنید.

نکات مهم​

  • همیشه قبل از نصب پکیج‌ها با pkg update && pkg upgrade مخازن و بسته‌های نصب شده را به‌روز نگه دارید.
  • دقت کنید که مجوزهای لازم به Termux داده شده باشد تا به حافظه و فایل‌ها دسترسی داشته باشد.

سوالات متداول (FAQ)​

1- Termux چیست و چرا باید از آن استفاده کنم؟
ج: Termux یک ترمینال لینوکسی سبک برای اندروید است که به شما امکان می‌دهد محیط توسعه لینوکسی را روی موبایل خود داشته باشید.

2- آیا نصب Node.js روی Termux کار سختیه؟
ج: خیر، با چند دستور ساده که در مقاله گفته شده نصب و راه‌اندازی می‌شود.

3- آیا می‌توانم بدون اینترنت کدنویسی کنم؟
ج: بله، پس از نصب تمام ابزارهای لازم، می‌توانید به صورت آفلاین هم کار کنید.
