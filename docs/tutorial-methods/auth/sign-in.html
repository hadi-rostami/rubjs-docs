signIn | ورود با کد تایید

این متد تابعی از کلاس Client است که برای ورود کاربر با کد تایید ارسال شده استفاده می‌شود.

توضیح کلی​

متد signIn داده‌های لازم برای تایید هویت کاربر شامل کد تایید، شماره تلفن، هش کد و کلید عمومی را به سرور ارسال می‌کند. این متد از تابع عمومی builder برای ارسال درخواست رمزنگاری‌شده استفاده می‌کند و از سشن موقت بهره می‌برد.

ورودی‌ها​

نام پارامترنوعتوضیح
phone_codestringکد تاییدی که به شماره تلفن ارسال شده است.
phone_numberstringشماره تلفن کاربر که کد تایید برای آن ارسال شده.
phone_code_hashstringهش مربوط به کد تایید که از قبل دریافت شده است.
public_keystringکلید عمومی رمزنگاری برای ارتباط امن با سرور.

خروجی​

در صورت موفقیت، این متد یک Promise از نوع SendCode بازمی‌گرداند.

interface SignIn {
  status: string;
  auth: string;
  user: {
    user_guid: string;
    first_name: string;
    last_name: string;
    phone: string;
    username: string;
    avatar_thumbnail: {
      file_id: string;
      mime: string;
      dc_id: string;
      access_hash_rec: string;
    };
    last_online: number;
    bio: string | boolean;
    is_deleted: boolean;
    is_verified: boolean;
    online_time: {
      type: string;
      exact_time: number;
    };
  };
  timestamp: number;
}

🔗 برای جزئیات بیشتر به صفحه پاسخ‌های مربوط به احراز هویت مراجعه کنید.​
